CONSOLE

Tools for running a display wall.

Arrange layouts, switch scenes, and monitor every panel from one place.

⬡

Add a machine, its screens show up

Run one agent on any PC or NUC. It registers the machine, discovers every connected display, and brings each one online as a first-class screen — named, addressable, and ready for content. No per-screen setup.

$ polyptic agent up
reception-pc online
Display 1 · 1920×1080 → screen
Display 2 · 1920×1080 → screen

HOW IT WORKS

Three steps to set up.

01

Connect your machines

Install the agent on each PC or NUC. Every output it drives appears as a screen.

02

Arrange the wall

Drag screens onto the canvas and align them edge-to-edge into murals that span content across panels.

03

Assign content

Assign dashboards, video, or web pages to screens. Save layouts as scenes to switch between them.

OPEN SOURCE

Polyptic is MIT-licensed and runs on your own hardware. There is no cloud service, no telemetry, and no per-seat pricing. The source is available to read, fork, and modify.
